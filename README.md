# PRODIGY_DS_02

## Project Overview
This project is part of the Prodigy Data Science tasks. It focuses on data cleaning, exploratory data analysis (EDA), and visualization to uncover patterns and insights from structured datasets.

## Objectives
- Clean and preprocess datasets
- Handle missing values and duplicates
- Perform exploratory data analysis (EDA)
- Visualize distributions, relationships, and trends

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook
